KnockoutJS Context Debugger

2013-04-12
Uncategorized

Have you seen the Chrome Extension that helps you with Knockout development? It adds a section to the Elements tab in the development tools that shows you the current binding context for any element you select on the DOM. This is a great tool to have when developing with Knockout.

https://chrome.google.com/webstore/detail/knockoutjs-context-debugg/oddcpmchholgcjgjdnfjmildmlielhof
